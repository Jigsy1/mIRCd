* THE FOLLOWING MODES ARE GENERALLY RESERVED AND THEREFORE SHOULD NOT BE USED.

Chanmodes:
----------
a - Admin on non ircu IRCds.
e - Ban exemption for n!u@h. (Some ircu based IRCds.)
q - Owner on non ircu IRCds or quiet on other non ircu IRCds. (n!u@h)
r - Only registered users can join the channel. (ircu)
I - Invite exemption for n!u@h on non ircu IRCds.
M - Only registered users may talk. (Some ircu based IRCds.)
Q - Quiet on some non ircu IRCds. (n!u@h)
Z - Only users on an SSL connection may join the channel. (Some ircu based IRCds.)

Usermodes:
----------
r - User is registered. (This is a hidden mode. Cannot be unset.) (ircu)
R - Only registered users can message you. (ircu)
W - Web clients. (Some ircu based IRCds.)
Z - User is connected via SSL. (Cannot be unset.) (Some ircu based IRCds.)

Deprecated usermodes:
-------------------------
O - Local IRC Operator.
